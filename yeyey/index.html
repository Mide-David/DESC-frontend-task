INTRODUCTION TO FRONTEND WEB DEVELOPMENT

frontend web development is a branch of full web development which involves creating, building and maintaining websites that run online on a browser.

What is Frontend web development?

Front end web development involves the "client-facing" side of a website. That is to say, frontend web development deals with the portion of the website, app, or digital product That users will interract with. It includes everything that users will interract with. It includes everyting the user will experience directly from text to columns, buttons, images, etc. To write the frontend of a webpage, different languages can be employed such as HTML, CSS, JAVASCRPIT, etc. Oter frontend tools may include; bootstrap, Angular, express, etc.

HTML

HTML stands for hyper-text mark up language. It is a text based approach to describing how contents contained witin an html file is structured. It is a standard markup language for web development. Html is basically used to structure contents (Texts, images, buttons, etc) on a webpage.

Html can be used in web development, internet navigation, web documentation, etc.

How HTML works

The average website includes different html pages which are usually contained in different html files or documents. Html documents end with a .html or.htm extension. A web browser reads te file and renders its content so that internet users can view it.

All html pages have a series of html elements, consisting of a set of tags and attributes. The three main parts of an element inhtml are;

  • opening tag: Which is used to state where the element is to begin.
  • content: this is the output that the user will see.
  • closing tag: this denotes the end of the elements.

HTML Tags

Every Html document must start with a "!Doctype" declaration to inform the web browser about the document type. Oter Html tags include;

  • Doctype tag
  • html tag
  • head tag
  • body tag
  • paragraph tag (p)
  • Strong tag (Strong)
  • Em tag (em)
  • small tag (small)
  • heading tag (h)
  • unordered list (ul)
  • ordered list (ol)
  • list tags (li)
  • div tags (div)
  • span tag (span)

    • Note that the aforementioned tags usually have a opening and closing tag, however, some tags are self closing and therefore do not make use of a closing tag, such tags include;

    1. Break tag (br)
    2. Horizontal rule (hr)
    3. image tag (img)
    4. Block quotes, etc.

CSS

Note that CSS can be added to an html document using the "style" tag or a separate file could be created in the document folder with the extension ".css" where the set of CSS rules are outlined.

CSS is an abbreviation that stands for "Cascading Style Sheet". It is used alongside html to make contents in a webpage look more presentable, in other words, CSS is used to style webpages. CSS makes use of stylesheets which are a list of different CSS rules or rule set used to style our html document.

CSS can be used to perform different operations such as; formatting background color, padding, line height, font size and font style, text decoration, text alignment, letter spacing coloumn count, border width, border color, text sadow, etc.

To format or design html documents, CSS uses a list of selectors such as;

  • Id selector
  • class selector
  • Descending selectors, etc.

In frontend web development, other programming languages such as; JAVASCRPIT, Tailwind, etc, are used to add the needed interractivity and behaviour to the webpages. They are used to enance user experience, create web servers and server-side applications, real time update of web pages, etc.

Asides Html, CSS, and JS, frontend web development also requires the knowledge of certain tools and libraries such as;

  • Bootstrap
  • SASS
  • jQuery
  • React
  • Version control with Git, etc.
